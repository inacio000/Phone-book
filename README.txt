# Phone-book
I created a telephone book.

 First I solved this exercise bellow, then I had other ideas and I was adding other options that are useful, likeÇ
 [1]-Add contact 
 [2]-Delete contact
 [3]-Show contact list 
 [4]-Close the telephone book 

Create an AGENDA program that stores to a file
“contatos.txt” names and telephone numbers of persons as given to
follow:

Name      Phone Number
Pedro     34567869
Rafael    58762001
Luis      76456002

To do so, create a loop that reads names and phone numbers until “exit” is typed.
when prompted for the name to be entered. Then print on the screen and on the
file “contatos.txt” each of the information read.

---------------------------------------------------------------------------

Criar um programa AGENDA que armazena em um arquivo
“contatos.txt” nomes e números de telefone de pessoas tal como dado a 
seguir: 
Nome    Telefone 
Pedro   34567869 
Rafael  58762001 
Luis    76456002 
Para tanto, crie um laço que lê nomes e telefones até que seja digitado “sair” 
quando for requisitado que o nome seja digitado. Depois imprima na tela e no 
arquivo “contatos.txt” cada uma das informações lidas.
